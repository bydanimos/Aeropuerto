--------------------------- Datos relevantes acerca de la implementación ---------------------------

Hemos utilizado un calendario adicional llamado JCalendar. Para importarlo al proyecto es necesario
abrir un JPanel o un JFrame para que a la derecha te aparezcan las diferentes opciones de SWing. Una
vez vemos estos elementos lo que hacemos es click derecho para añadir uno nuevo, pulsamos la opción de
'Palette Manager', donde creamos una nueva con el nombre que queremos. Una vez creado le damos a la 
opción de 'ADD from Jar' donde seleccionamos el ejecutable de nuestra ruta de directorios y añadimos 
los ocho tipos que trae.

Es recomendable tener el '.jar' dentro del proyecto.

En caso de erro añadir el ejecutable de la misma manera que se añade el gestor de la base de datos
'postgresql.jar'

Una vez añadido ya está listo para ser utilizado.



Decir también que hemos experimentado un error con el tamaño del panel en la ventana de estadísticas, 
este error no se nos manifiesta en macOS, pero sí en algunas ocaciones en Ubuntu, tanto en la versión
18.04 LTS como la 20.04 LTS, aunque no siempre.
